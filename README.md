# T-Библиотека

Консольное приложение для управления личной библиотекой книг. Написано на Python.

## Как запустить

```
python library.py
```

Нужен Python 3.10 или выше.

## Что умеет

- Добавлять книги (название, автор, жанр, год, описание)
- Просматривать список с сортировкой и фильтрами
- Искать книги по ключевым словам
- Отмечать книги как прочитанные
- Добавлять в избранное
- Редактировать и удалять книги
- Показывать статистику

## Хранение данных

Все книги сохраняются в файл `tlib_data.json` автоматически. При следующем запуске данные загружаются из него же.

## Структура проекта

```
library.py       — основной файл, запускать его
tlib_data.json   — создаётся автоматически после добавления первой книги
```
